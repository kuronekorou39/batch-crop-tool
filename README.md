# Batch Crop Tool

複数の画像を一括で切り抜きできるPySide6製のGUIツール。一度設定した切り抜き範囲を複数の画像に同時適用できます。

## 🎯 主な機能

- **一括切り抜き処理** - 一つの画像で設定した切り抜き範囲を複数画像に適用
- **直感的なマウス操作** - ドラッグで範囲選択、ハンドルで微調整
- **リアルタイム座標表示** - 切り抜き範囲の座標をリアルタイムで確認
- **2つの切り抜きモード**
  - 絶対座標モード：同じサイズの画像に同じ座標で切り抜き
  - 比率モード：異なるサイズの画像でも相対的な位置で切り抜き
- **柔軟なファイル管理**
  - ドラッグ&ドロップ対応
  - 複数選択・個別削除
  - 右クリックメニュー

## 📋 必要要件

- Python 3.8以上
- PySide6

## 🚀 インストール

1. リポジトリをクローン
```bash
git clone https://github.com/kuronekorou39/batch-crop-tool.git
cd batch-crop-tool
```

2. 仮想環境を作成
```bash
python -m venv venv
```

3. 仮想環境を有効化

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

4. 依存パッケージをインストール
```bash
pip install -r requirements.txt
```

## 💻 使い方

1. 仮想環境を有効化（まだの場合）
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

2. ツールを起動
```bash
python image_batch_cropper.py
```

2. 画像を追加
   - 「画像を追加」ボタンから選択
   - またはファイルをドラッグ&ドロップ

3. 切り抜き範囲を設定
   - マウスドラッグで範囲を選択
   - 赤いハンドルをドラッグして微調整
   - または座標を数値で直接入力

4. 切り抜きを実行
   - 「選択範囲で切り抜き実行」をクリック
   - 必要に応じて切り抜きモードを選択

5. 保存
   - 「切り抜いた画像を保存」で保存先を指定

## 🎮 操作方法

### マウス操作
- **左クリック + ドラッグ**: 新規範囲選択
- **範囲内をドラッグ**: 範囲を移動
- **角のハンドルをドラッグ**: 対角方向にリサイズ
- **辺のハンドルをドラッグ**: 水平/垂直方向にリサイズ

### キーボードショートカット
- **Ctrl/Shift + クリック**: 複数ファイル選択
- **右クリック**: コンテキストメニュー表示

## 🔧 切り抜きモード

### 絶対座標モード
同じサイズの画像に対して、まったく同じピクセル座標で切り抜きます。
ブラウザのスクリーンショットなど、同一サイズの画像処理に最適。

### 比率モード
画像サイズが異なっても、相対的な位置とサイズで切り抜きます。
異なる解像度の画像でも同じような領域を切り抜きたい場合に便利。

## 📝 対応画像形式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- BMP (.bmp)
- GIF (.gif)
